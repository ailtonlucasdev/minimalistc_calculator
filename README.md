# Calculadora Minimalistica Online
Uma calculadora simples usando html, css e javascript

Projeto de uma calculadora simples funcional desenvolvidada com HTML5, CSS3 e JavaScript.

## Fucionalidades
- Operações matemáticas básicas (+, -, x, ÷);
- Operações matemáticas avançadas, como porcentagem, raiz quadrada e potenciação;
- Tratamento de erros gerados por expressões inválidas.

## Estrutura 
- `index.html`: base da calculadora;
- `style.css`: estilização e layout da calculadora;
- `script.js`: funcionalidades e lógica dos cálculos.

## Rodar o projeto
- Pode clonar o repositório e abrir e o `index.html` no seu navegador
- Acessar diretamente pelo link:
  https://minimalistc-calculator.vercel.app/
